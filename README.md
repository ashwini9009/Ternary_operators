# Ternary Operator Practice in Java

This repository contains a simple Java program created to practice ternary operators.  
Ternary operator is a short hand for if- else.
All basic conditional problems are solved using the Ternary (? :) operator instead of if-else.

The program is written in a single Java file: Test.java.

Technologies Used
- Java
- JDK (Core Java concepts)

 Problems Solved Using Ternary Operator
The following basic practice questions are implemented:

1. Maximum of two numbers  
2. Minimum of two numbers  
3. Check whether a number is Positive or Negative
4. Check whether a number is Even or Odd  
5. Maximum of three numbers  
6. Check whether a year is a Leap Year or Not 
7. Check whether a number is 2-digit or 3-digit
